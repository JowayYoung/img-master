> `img-master`是一个多进程无限制的图片压缩工具

### 安装

`npm i -g img-master`

> 安装失败

- 将npm源镜像设置为淘宝镜像：`npm config set registry https://registry.npm.taobao.org`
- 重新执行命令安装：`npm i -g img-master`

### 使用

命令|缩写|功能
:-:|:-:|-
`img-master compress` | `img-master c` | 压缩图片
